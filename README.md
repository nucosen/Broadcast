## Tags

`niconico` `nicolive` `nico-nico-douga`

## Advantages

-   自動枠取り
-   REST APIを備えたデータベースでリクエスト受付ができる
-   枠の終了時間を確認し、音楽が途切れそうならば次枠に繰り越す

## Installation

Pipを用いてリリースのtar.gzファイルを展開してください。

Python3.10以降必須

## Minimal Example

`nucosen`コマンドで起動します
